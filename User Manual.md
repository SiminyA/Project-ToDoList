## User Manual
#### Purpose 
Provide the instructional manual for the users to enable the user to use the toDoly applications 
#### Instructions
1. Search for Terminal in MAC machine
2. Go to the specific folder the application resides (using cd command)
3. Type javac MainClass.java
4. Type java MainClass
5. User will be displayed with ToDoLy menu

<img width="447" alt="mainmenu" src="https://user-images.githubusercontent.com/29890115/46719535-f9ff1b80-cc22-11e8-8e03-cdbf0c3dd7ee.png">

6. User select "1" from the above displayed menu 

<img width="617" alt="show by date" src="https://user-images.githubusercontent.com/29890115/46722426-4c900600-cc2a-11e8-9a94-e15cee66f7ba.png">

7. User will be displayed with the tasks based on the duedate

<img width="617" alt="  show by date" src="https://user-images.githubusercontent.com/29890115/46734631-bae3c100-cc48-11e8-951c-49c8f36a85df.png">

8. User can also view the tasks based on project name

<img width="694" alt="show task by project" src="https://user-images.githubusercontent.com/29890115/46734828-39406300-cc49-11e8-8a45-12383be2b79b.png">

9. User will select '2' from the ToDoly Menu to add the task 
10. User will enter the new taskname , status & Project Name

<img width="422" alt="add task" src="https://user-images.githubusercontent.com/29890115/46735102-13678e00-cc4a-11e8-924e-a323b9481642.png">

11. The new task will be added to the file
12. User to select option '3' to edit the tasks. User will displayed with all the tasks added 
13. User can specify the specific taskno to perform the edit operations.

<img width="828" alt="edit task" src="https://user-images.githubusercontent.com/29890115/46735504-3e9ead00-cc4b-11e8-834b-acd51329184f.png">

14. User selects to edit the entire task details

<img width="642" alt="edit task new values" src="https://user-images.githubusercontent.com/29890115/46735837-4874e000-cc4c-11e8-9dce-4f0fa64120fc.png">

15. User can choose to update the status of the Task as Done

<img width="419" alt="edit - mark done" src="https://user-images.githubusercontent.com/29890115/46745464-f048d880-cc61-11e8-8623-65e786c04f4d.png">

16. User can also choose to delete the task

<img width="426" alt="delete task" src="https://user-images.githubusercontent.com/29890115/46745534-1c645980-cc62-11e8-8035-b159246bbb4e.png">

17. User to select option '4'. 

<img width="487" alt="save and exit" src="https://user-images.githubusercontent.com/29890115/46745601-39009180-cc62-11e8-9f91-2465e0e03e69.png">

18. The tasks will be added and saved to the file
19. The user will be exited from the menu interface
