# User Manual

#### 1. Purpose 
The purpose of this document is to provide detailed instruction on how a new user can use the ToDoly application to perform various such as add new tasks,
update the tasks , delete tasks , mark the specific tasks as done.

#### 2. Instructions

##### 1. Import the code from the GitHub repository

##### 2. Open the project from IntelliJ IDEA

##### 3. click Run -> ToDoListMain

##### 4. User will be displayed with ToDoLy menu

<img width="617" alt="  show by date" src="https://user-images.githubusercontent.com/29890115/46734631-bae3c100-cc48-11e8-951c-49c8f36a85df.png">

<img width="694" alt="show task by project" src="https://user-images.githubusercontent.com/29890115/46734828-39406300-cc49-11e8-8a45-12383be2b79b.png">
